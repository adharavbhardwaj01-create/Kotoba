# Kotoba

A free, open-source Japanese language learning app powered by AI.

Learn Japanese through interactive lessons, vocabulary decks, speech recognition, and conversation practice. Bring your own API key — no paywalls, no premium tiers.

## Features

- Vocabulary decks with spaced repetition
- Speech recognition and pronunciation practice
- AI-powered conversation scenarios
- Hiragana & Katakana reference
- Progress tracking with daily streaks
- Mobile-first responsive design

## Getting Started

1. Clone the repo
2. Install dependencies: `npm install`
3. Copy `.env.example` to `.env` and add your Gorp API key
4. Run the dev server: `npm run dev`

## Tech Stack

- React 19 + Vite
- Gorp API (BYOK — Bring Your Own Key)
