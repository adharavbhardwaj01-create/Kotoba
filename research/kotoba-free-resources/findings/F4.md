## 1. Web Speech API

| Attribute | Details |
|-----------|---------|
| **Name** | Web Speech API (SpeechSynthesis) |
| **URL** | https://developer.mozilla.org/en-US/docs/Web/API/Web_Speech_API |
| **Japanese Voices** | Uses OS/browser built-in voices; quality varies by platform. Chrome/Edge typically provide 2-3 Japanese voices (e.g., Google Japanese, Microsoft Haruka). Safari uses macOS/iOS built-in voices. |
| **Voice Quality** | Medium. Depends entirely on OS/browser. Chrome's Google Japanese voice is decent but not natural. iOS/macOS voices are higher quality. |
| **Free Tier Limits** | Completely free, unlimited usage. No API keys required. |
| **Latency** | Low (~50-200ms). Runs locally on device. |
| **Ease of Integration** | Very Easy. Simple JavaScript API. No server setup needed. |
| **Browser Support** | Chrome 33+, Edge 79+, Safari 7+, Firefox 49+ |
| **Pros** | Zero cost, no API keys, works offline, instant setup |
| **Cons** | Voice quality varies by OS, limited voice selection, no SSML support, inconsistent cross-browser behavior |
| **Recommendation** | Good for MVP/prototyping. Use as fallback for offline mode. |

## 2. Google Cloud Text-to-Speech

| Attribute | Details |
|-----------|---------|
| **Name** | Google Cloud Text-to-Speech |
| **URL** | https://cloud.google.com/text-to-speech |
| **Japanese Voices** | Multiple Japanese voices available: Standard (ja-JP-Standard-A/B/C/D), WaveNet (ja-JP-Wavenet-A/B/C/D), Neural2 (ja-JP-Neural2-B/C/D), Chirp 3 HD voices. Both male and female options. |
| **Voice Quality** | High. WaveNet voices are very natural. Neural2 voices are excellent. Chirp 3 HD is top-tier quality. |
| **Free Tier Limits** | WaveNet/Standard: 4 million characters/month free. Neural2: 1 million characters/month free. Chirp 3 HD: 1 million characters/month free. |
| **Latency** | Low-moderate (~100-300ms for neural, ~200-500ms for WaveNet) |
| **Ease of Integration** | Moderate. Requires Google Cloud account, API key/service account. SDKs available for web, mobile, Python, Node.js, etc. |
| **Post-Free Tier Pricing** | Standard/WaveNet: $4/1M chars. Neural2: $16/1M chars. Chirp 3 HD: $30/1M chars |
| **Pros** | Excellent Japanese voice quality, generous free tier, SSML support, streaming available |
| **Cons** | Requires billing enabled even for free tier, complex pricing tiers |
| **Recommendation** | Strong option for production use. WaveNet voices offer best value. |

## 5. Open-Source TTS Alternatives

### 5a. Coqui TTS (XTTS v2)

| Attribute | Details |
|-----------|---------|
| **Name** | Coqui TTS |
| **URL** | https://github.com/coqui-ai/TTS |
| **Japanese Voices** | Supports 16+ languages including Japanese via XTTS v2. Can use voice cloning with any reference audio. |
| **Voice Quality** | Good-High. XTTS v2 produces natural-sounding speech. Voice cloning can match reference quality. |
| **Free Tier Limits** | Completely free and open-source (MPL-2.0 license). No usage limits. |
| **Latency** | Moderate-High (~500ms-2s depending on hardware). Requires GPU for real-time. |
| **Ease of Integration** | Complex. Requires Python, PyTorch, model downloads (~2GB). Self-hosting required. |
| **Pros** | Free forever, voice cloning, multi-language, no API limits, customizable |
| **Cons** | Requires significant compute resources (GPU recommended), complex setup, not production-ready out of box |
| **Recommendation** | Good for offline/desktop apps or if you need voice cloning. Not ideal for web/mobile without backend. |

## Recommendation for Kotoba

### Tier 1: Primary Choice
**VOICEVOX** - Best Japanese voice quality, completely free, offline-capable, 40+ character voices ideal for language learning variety.

### Tier 2: Cloud Backup
**Azure TTS** - Excellent Japanese voices with speaking styles, generous free tier (0.5M chars/mo), good for web deployment.

### Tier 3: Simple Fallback
**Web Speech API** - Zero setup, works everywhere, good for quick prototyping.

### Implementation Strategy
1. **MVP Phase**: Use Web Speech API for rapid development
2. **Beta Phase**: Integrate VOICEVOX for high-quality Japanese audio
3. **Production Phase**: 
   - Primary: VOICEVOX (offline mode)
   - Cloud: Azure TTS or Google Cloud TTS (for web/when offline not available)
   - Fallback: Web Speech API
