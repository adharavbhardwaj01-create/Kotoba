## 1. Web Speech API (SpeechRecognition)

**URL**: https://developer.mozilla.org/en-US/docs/Web/API/Web_Speech_API

**Japanese Support**: Browser-dependent
- Chrome: Uses Google's servers, good Japanese support
- Safari: Uses Apple's servers, Japanese support varies
- Firefox: Limited/no speech recognition support

**Accuracy**: Moderate (depends on browser backend)
- Chrome typically uses Google's speech engine (good accuracy)
- No control over recognition quality

**Free Tier**: Unlimited - browser-native, no API keys needed

**Offline Capability**: 
- Newer Chrome versions support on-device speech recognition
- Requires user to enable in Chrome settings
- Generally requires internet connection

**Ease of Integration**: 
- Very easy - pure JavaScript, no API keys
- Simple API: `webkitSpeechRecognition` or `SpeechRecognition`
- Example:
```javascript
const recognition = new (window.SpeechRecognition || window.webkitSpeechRecognition)();
recognition.lang = 'ja-JP';
recognition.start();
```

**Limitations**:
- Browser-dependent behavior
- No control over recognition engine
- Limited to short utterances
- No pronunciation assessment features

## 2. Google Cloud Speech-to-Text

**URL**: https://cloud.google.com/speech-to-text

**Japanese Support**: 
- Full support with `ja-JP` locale
- Models: `chirp` and `chirp_2` (latest)
- Supports automatic punctuation for Japanese

**Accuracy**: High (state-of-the-art)
- Chirp 2 model offers excellent accuracy
- Supports model adaptation for custom vocabularies

**Free Tier**:
- V1 API: **60 minutes/month free** (with data logging)
- V1 API: **60 minutes/month free** (without data logging)
- New accounts get $300 free credits
- After free tier: $0.016/minute

**Offline Capability**: No (cloud-only)

**Ease of Integration**:
- REST API and client libraries (Python, Node.js, etc.)
- Streaming and batch transcription
- Requires Google Cloud account and API key

**Key Features**:
- Automatic punctuation
- Speaker diarization
- Model adaptation
- Word-level confidence

## 3. Azure Speech Services

**URL**: https://azure.microsoft.com/en-us/products/ai-foundry/tools/speech/

**Japanese Support**: 
- Full support with `ja-JP` locale
- Fast transcription supported
- Custom speech available for Japanese

**Accuracy**: High (enterprise-grade)
- Neural models with good accuracy
- Custom speech training available

**Free Tier (F0)**:
- **5 audio hours/month** for Speech-to-Text
- 0.5 million characters/month for Text-to-Speech
- 5 audio hours/month for Speech Translation
- No credit card required for free tier

**Offline Capability**: 
- Connected containers available (paid)
- Disconnected containers for air-gapped scenarios

**Ease of Integration**:
- Speech SDK available for multiple platforms
- REST API
- JavaScript/Web Speech SDK available
- Requires Azure subscription (free tier available)

**Key Features**:
- Pronunciation assessment (excellent for language learning!)
- Custom speech models
- Real-time and batch transcription
- Multiple neural voices for Japanese TTS

## 4. Whisper (OpenAI)

**URL**: https://github.com/openai/whisper

**Japanese Support**: 
- Excellent - multilingual model
- Dedicated language detection
- High accuracy for Japanese

**Accuracy**: Very High
- large-v3 model: ~8.4% CER (character error rate) on Japanese
- Best open-source accuracy for Japanese

**Free Tier**: 
- **100% free** - MIT License
- No usage limits
- Self-hosted, no API costs

**Offline Capability**: Yes (fully offline)

**Model Sizes**:
| Model | Parameters | VRAM Required | Speed |
|-------|-----------|---------------|-------|
| tiny | 39M | ~1 GB | ~10x |
| base | 74M | ~1 GB | ~7x |
| small | 244M | ~2 GB | ~4x |
| medium | 769M | ~5 GB | ~2x |
| large | 1550M | ~10 GB | 1x |
| turbo | 809M | ~6 GB | ~8x |

**Ease of Integration**:
- Python library: `pip install openai-whisper`
- Simple API:
```python
import whisper
model = whisper.load_model("turbo")
result = model.transcribe("audio.mp3", language="ja")
```
- Requires ffmpeg
- GPU recommended for real-time

**Key Features**:
- Automatic language detection
- Timestamps for each word
- Translation to English available
- No internet required

## 5. Vosk

**URL**: https://alphacephei.com/vosk/

**Japanese Support**: Yes - dedicated models available

**Japanese Models**:
| Model | Size | CER | Notes |
|-------|------|-----|-------|
| vosk-model-small-ja-0.22 | 48 MB | 9.52 (csj), 17.07 (ted10k) | Lightweight for mobile |
| vosk-model-ja-0.22 | 1 GB | 8.40 (csj), 13.91 (ted10k) | High accuracy server |

**Accuracy**: Good
- Small model: ~9-17% CER
- Large model: ~8-14% CER
- Not as accurate as Whisper or cloud services

**Free Tier**: 
- **100% free** - Apache 2.0 License
- No usage limits
- Self-hosted

**Offline Capability**: Yes (fully offline)

**Ease of Integration**:
- Python: `pip install vosk`
- JavaScript/Node.js bindings available
- Java, C#, and other language bindings
- Streaming API for real-time
```python
from vosk import Model, KaldiRecognizer
import soundfile as sf

model = Model("vosk-model-small-ja-0.22")
rec = KaldiRecognizer(model, 16000)
```

**Key Features**:
- Very lightweight (48MB for small model)
- Runs on Raspberry Pi, mobile devices
- Dynamic vocabulary reconfiguration
- Speaker identification

**Limitations**:
- Lower accuracy than Whisper or cloud services
- Requires audio in specific format (16kHz PCM)

## 8. Browser-based vs Cloud-based Comparison

| Aspect | Browser-based (Web Speech API) | Cloud-based (Google/Azure) | Self-hosted (Whisper/Vosk) |
|--------|-------------------------------|---------------------------|---------------------------|
| **Cost** | Free | Free tier + paid | Free (compute costs) |
| **Offline** | Limited | No | Yes |
| **Accuracy** | Moderate | High | High (Whisper) / Good (Vosk) |
| **Latency** | Low (server-dependent) | Medium (network) | Low (local) |
| **Privacy** | Data sent to browser vendor | Data sent to cloud | Data stays local |
| **Setup** | None | API key required | Installation required |
| **Scalability** | Limited | Excellent | Hardware-dependent |
| **Japanese Quality** | Variable | Excellent | Good-Excellent |

## Recommendations for Kotoba

### Primary Recommendation: **Whisper (self-hosted)**
- Best accuracy for Japanese
- Completely free with no limits
- Offline capability (important for mobile)
- Active development and community

### Secondary Recommendation: **Azure Speech Services**
- Free tier: 5 hours/month (sufficient for learning app)
- Excellent pronunciation assessment feature (perfect for language learning!)
- Easy JavaScript SDK integration
- Enterprise reliability

### Quick Start Option: **Web Speech API**
- Zero setup for browser-based version
- Good enough for basic features
- Can upgrade to Whisper/Azure later

### Budget-Friendly Offline: **Vosk**
- Tiny model size (48MB) for mobile
- Good enough accuracy for learning apps
- No cloud dependency
